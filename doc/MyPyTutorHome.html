MyPyTutor Home

MyPyTutor is an interactive tutorial system for Python. It's main use is to provide tutorials for students in a Python related course.

The main features of MyPyTutor include:
  • IDLE-like code edit window
  • Automatic checking of answers with good feedback on errors
  • Optional online support for submitting answers and collecting results
  • Support for course adminstrators and tutorial builders
